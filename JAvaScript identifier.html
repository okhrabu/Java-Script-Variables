Java Script Identifier

All JavaScript varable must be identified with unique names.
These unique name are called identifiers
Identifiers can be shorts name (like x and Y) or more discriptive names (age,sum,total volume)
The geneal rules for constructing names for variables (unique identifiers) are:

    -Name can contain letters digits underscore and dollar signs.
    -Name must begain with a letter.
    -Names can also begain with $ and _(but will be not use in it in this tutorial).
    -Name are case sensitive (Y and y are different variables ).
    -Reserved words (like JavaScript keywords)can not be use as names.

Note

JavaScript identifiers are case-sensitive.

The Assignment operator
In JavaScript, the qual sign (=) is an "assigment" operator, not an "equal to " operator.
This is different from algebra. The flowing does not make sense in algebra:

x = x + 5

In JavaScript, however, it makes perfect sense: it assigns the value of x + 5 to x. (It calculates the value of x + 5 and puts the result into x. The value of x is incremented by 5.)

Note

The "equal to" operator is written like == in JavaScript.
