Declearing a JavaScript variable.

Creating a variable in JavaScript is called declearing a variable.

You declear a JavaScript variable with the or the let Keyword:

    var carName;

    or

    let carName;

After the decleartion,the variable has no value (technically it is undifined).

To assign a value to the variable,use the equal sign:
carName = "Volvo";

You can asign a value to thevariable when you declear it:
let carName = "Volvo";
